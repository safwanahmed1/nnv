
 Exit Games Photon Android SDK Overview
 ---------------------------------------
 
 Please refer to the index.html page in the documentation provided in the folder sdk/doc 
 for information on how to integrate the Photon library in your Android application.

 Please note that you need Android SDK 1.5 (or higher) to build the Demo projects in this SDK.
 
 Feel free to mail us: developer@exitgames.com  
 

 SDK contents
 ------------
 
demo-realtime		This simple demo shows how to use Photon and the Photon Lite Application to send positional updates.
					A compatible version of this application exists in each of the Photon client SDKs.
libs				Photon library file to be included in your application
doc					API Reference Documentation
